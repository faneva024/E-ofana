# Init Vue3 App (example scaffold)

Quick start:

```bash
npm install
npm run dev
```

Notes:
- This scaffold includes Vue 3, Vue Router, Pinia, Axios and Bootstrap 5 (SCSS).
- API endpoints in `src/api/axios.js` point to `http://localhost:3000/api` by default.
- Auth is stored in `localStorage` for the example; replace with secure storage as needed.
